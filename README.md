## 如何在本地使用  
你应该在本地运行一个静态页面服务端  
琴梨梨自用方法：  
先运行此命令安装    
`npm install --global http-server`  
然后使用此命令启动  
`http-server`  
之后在浏览器内访问`127.0.0.1:8080`即可  
特别注意：为正常调试Service Worker你必须在浏览器内使用127.0.0.1或localhost访问，否则会因为不安全的域无法安装
